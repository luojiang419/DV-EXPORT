# 测试清单

## 已自动验证

- 命名模板 token 展开
- Windows 文件名非法字符清洗
- Render Settings 映射
- 批量导出任务计划生成
- Ctrl / Shift 多选逻辑
- TypeScript 类型检查
- Vite 生产构建

## 建议在 Resolve 中手动验证

- 启动插件后能正确识别 Resolve Studio 19+
- 能展示媒体池文件夹树
- 选择某个文件夹后，只显示该文件夹内的时间线
- Ctrl 多选、Shift 连选行为符合预期
- 切换格式后编码器列表联动更新
- 选择输出目录后能够创建多条 render job
- 批量导出时不会清空原有 Render Queue
- 导出失败时能返回逐条结果说明

## 特殊场景

- 空文件夹
- 同名时间线
- 未选择输出目录直接点击导出
- Resolve 已在渲染中时点击导出
- 预设加载失败
