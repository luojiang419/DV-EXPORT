# v0.1.25 发布说明

## 给用户发送的文件

优先发送以下两个文件：

```text
DV-EXPORT-v0.1.25-setup.exe
DV-EXPORT-v0.1.25-setup.sha256.txt
```

如果用户电脑拦截 EXE，发送备用 ZIP：

```text
DV-EXPORT-v0.1.25-windows-installer.zip
DV-EXPORT-v0.1.25-windows-installer.sha256.txt
```

## 用户安装步骤

1. 关闭 DaVinci Resolve Studio。
2. 双击 `DV-EXPORT-v0.1.25-setup.exe`。
3. 弹出管理员权限确认时选择允许。
4. 安装完成后重新打开 DaVinci Resolve Studio。
5. 在 Resolve 菜单打开：

```text
Workspace -> Workflow Integrations -> 达芬奇批量导出
```

## ZIP 备用安装步骤

1. 解压 `DV-EXPORT-v0.1.25-windows-installer.zip`。
2. 双击解压后的 `install-windows.bat`。
3. 弹出管理员权限确认时选择允许。
4. 安装完成后重新打开 DaVinci Resolve Studio。

## 用户运行要求

- Windows
- DaVinci Resolve Studio 19 或更高版本
- 安装前必须关闭 DaVinci Resolve Studio

## 默认安装位置

```text
C:\ProgramData\Blackmagic Design\DaVinci Resolve\Support\Workflow Integration Plugins\com.dvexport.batch-export
```

安装器会自动备份旧版本插件，并安装当前版本。

## 发布前自检

- 确认 EXE 和 ZIP 文件名都是 `v0.1.25`
- 确认 `.sha256.txt` 与对应安装包一起发送
- 确认用户收到的是 `setup.exe`，不是 `setup-files` 临时目录
