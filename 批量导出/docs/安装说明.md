# 安装说明

## 1. EXE 自安装包

发布给普通用户时，优先使用 EXE 自安装包：

```powershell
cd G:\data\app\DV-EXPORT\批量导出
npm run build:installer
```

构建后会生成：

- `dist/installer/DV-EXPORT-vX.Y.Z-setup.exe`
- `dist/installer/DV-EXPORT-vX.Y.Z-setup.sha256.txt`
- `dist/installer/DV-EXPORT-vX.Y.Z-windows-installer.zip`
- `dist/installer/DV-EXPORT-vX.Y.Z-windows-installer.sha256.txt`

用户安装步骤：

1. 关闭 DaVinci Resolve Studio。
2. 双击 `DV-EXPORT-vX.Y.Z-setup.exe`。
3. 按提示允许管理员权限。
4. 安装完成后重新打开 DaVinci Resolve Studio。

安装器会自动备份旧版本插件，并安装到：

`C:\ProgramData\Blackmagic Design\DaVinci Resolve\Support\Workflow Integration Plugins\com.dvexport.batch-export`

如果 EXE 被安全软件拦截，可使用同目录生成的 ZIP 备用包：解压 `DV-EXPORT-vX.Y.Z-windows-installer.zip` 后双击 `install-windows.bat`。

## 2. 手动构建产物

仅开发或排查安装包问题时，手动构建插件目录：

```bash
cd /mnt/g/data/app/DV-EXPORT/批量导出
"/mnt/c/Program Files/nodejs/node.exe" ./scripts/build-plugin.mjs
```

构建后会生成：

- `dist/vX.Y.Z/com.dvexport.batch-export`

## 3. 手动安装到 Resolve

将整个 `com.dvexport.batch-export` 文件夹复制到：

`C:\ProgramData\Blackmagic Design\DaVinci Resolve\Support\Workflow Integration Plugins\`

## 4. WorkflowIntegration.node

如果构建产物中存在 `MISSING_WORKFLOWINTEGRATION_NODE.txt`，说明本机没有自动找到 `WorkflowIntegration.node`。

请从 Resolve Studio 自带开发目录复制：

`C:\ProgramData\Blackmagic Design\DaVinci Resolve\Support\Developer\Workflow Integrations\Examples\SamplePlugin\WorkflowIntegration.node`

复制到插件根目录：

`com.dvexport.batch-export\WorkflowIntegration.node`

自动安装包构建时会校验 `WorkflowIntegration.node` 是否存在；缺少该文件时不会生成可分发安装包。

## 5. 打开插件

在 DaVinci Resolve Studio 中进入：

`Workspace -> Workflow Integrations -> 达芬奇批量导出`
