# 安装说明

## 1. 构建产物

构建命令：

```bash
cd /mnt/g/data/app/DV-EXPORT/批量导出
"/mnt/c/Program Files/nodejs/node.exe" ./scripts/build-plugin.mjs
```

构建后会生成：

- `dist/vX.Y.Z/com.dvexport.batch-export`

## 2. 安装到 Resolve

将整个 `com.dvexport.batch-export` 文件夹复制到：

`C:\ProgramData\Blackmagic Design\DaVinci Resolve\Support\Workflow Integration Plugins\`

## 3. WorkflowIntegration.node

如果构建产物中存在 `MISSING_WORKFLOWINTEGRATION_NODE.txt`，说明本机没有自动找到 `WorkflowIntegration.node`。

请从 Resolve Studio 自带开发目录复制：

`C:\ProgramData\Blackmagic Design\DaVinci Resolve\Support\Developer\Workflow Integrations\Examples\SamplePlugin\WorkflowIntegration.node`

复制到插件根目录：

`com.dvexport.batch-export\WorkflowIntegration.node`

## 4. 打开插件

在 DaVinci Resolve Studio 中进入：

`Workspace -> Workflow Integrations -> 达芬奇批量导出`
